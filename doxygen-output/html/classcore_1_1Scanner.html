详细描述

High-level scanner managing scan sessions

The Scanner maintains:

  • m_matches: Current/active matches from the most recent scan
  • m_results: History queue of all saved scan results

auto core::Scanner::performFilteredScan ( const ScanOptions &  opts,
const UserValue *  value = nullptr,
bool  saveToHistory = false 
) -> std::expected<ScanStats, std::string>
inlineexport

Perform a filtered (incremental) scan only on previously matched addresses Narrows existing matches according to new scan options/value without rescanning all regions. Logic: For every byte previously marked as matched, re-read memory at that address and apply the selected scan routine. Bytes failing the new criterion are cleared. Empty swaths are pruned.

参数
optsNew scan options (dataType/matchType/etc.)
valueNew user value (optional)
saveToHistoryIf true, snapshot the narrowed result into history
返回
Expected updated stats or error string
